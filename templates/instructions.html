How to Use This Website

  

Langkah-langkah menggunakan website ini:

  1. Login terlebih dahulu pada rute / atau menu awal untuk mendapatkan token
  2. Untuk masuk menu pemilihan algoritma sorting gunakan rute /home?token={token yang didapat}
  3. Untuk menu pemilihan file, nomor kolom, dan orientasi rute, gunakan rute /sort/{namaalgortima}?token={token}
  4. Untuk melihat hasil sorting, gunakan rute /sort/result?token={token}&id={id yang ingin dipilih}
